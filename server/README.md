# Velocity Exchange backend module

## Installation and Boot command

To install prerequisites, run

```
npm ci
```

To boot server, run

```
npm start
```

Server is ready at _http://localhost:1111_

## Usage

### Sign Up

`/auth/sign-up/`

- email

- password

- username

- first_name

- last_name

- dob: "YYYY-MM-DD"

- phone_num

### Login

`auth/login/`

- email

- password
